用Python和C++实现卡机病毒---100个cmd窗口

2020-03-30 1 min read # C++ # 软件 # 电脑操作 # Python # 文件

上代码：
Python：

import os
#将下面改为"while 1:"会永远执行（直到叉掉或关机）
for x in range(1,100):
    os.system("start")

C++：

#include <windows.h>
using namespace std;
int main()
{
/*永远执行需要将下面一句改为
    "while(1)"*/
	for(int i = 1;i <= 100;i++)
	{
		system("start");
	}
	return 0;
}
